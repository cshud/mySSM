# mySSM

**SSM Demo架构**：Spring + SpringMVC +Mybatis

- Maven：项目管理和自动构建工具。

- Spring：Ioc容器，由DI来实现；支持AOP（事务处理，日志记录）。

- SpringMVC：Spring MVC控制器。Restful风格。

- Mybatis：mybatis-generator自动生成代码

- Quartz定时调度任务

- Ehcache缓存框架

- Druid数据库连接池：支持Druid内置监控。

- Mongodb：日志记录

- Mysql
